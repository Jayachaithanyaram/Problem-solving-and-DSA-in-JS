## ▶️ How to Run the Programs

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/repository-name.git
   ```

2. Navigate into the folder:

   ```bash
   cd repository-name
   ```

3. Run any file using Node.js:

   ```bash
   node filename.js
   ```

Example:

```bash
node factorial.js
```
